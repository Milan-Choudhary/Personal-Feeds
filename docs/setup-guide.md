# Setup Guide

## Prerequisites

- Node.js
- npm or yarn
- MongoDB

## Install

1. Navigate to the repository root.
2. Install dependencies for each package, for example:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   cd ../ai-engine && npm install
   ```

## Run

- Start backend: `cd backend && npm run dev`
- Start frontend: `cd frontend && npm run dev`

## Environment

- Configure root `.env` and package-specific `.env` files as needed.
