# AI Content Platform

Monorepo for a content creation platform with separate frontend, backend, and AI engine packages.

## Structure

- `frontend/` - React app with Vite
- `backend/` - Node + Express API
- `ai-engine/` - AI logic and prompts
- `shared/` - Shared utilities
- `docs/` - Documentation

## Getting Started

Create package files in each package and install dependencies as needed.
